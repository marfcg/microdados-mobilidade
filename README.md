<a id="user-content-microdados-mobilidade" class="anchor" href="#microdados-mobilidade" aria-hidden="true"><span class="octicon octicon-link"></span></a>microdados-mobilidade</h1>

<p>Extração de dados de mobilidade a partir dos microdados do Censo 2010</p>

<p>Algumas observações importantes:</p>

<ul>
<li>Para fins de cálculo de fluxo de mobilidade, deslocamento para estudo ou trabalho são considerados equivalentes, salvo quando há ocorrência de ambos;</li>
<li>Nos casos em que há tanto mobilidade para estudo quanto para trabalho, optou-se pelo destino do trabalho;</li>
<li>Há casos de pessoas que relatam mobilidade porém sem informação sobre seu destino. Tais ocorrências podem possuir
mais de um marcador na tabela final (ex.: "Em branco" ou "IGNORADO").</li>
</ul>
